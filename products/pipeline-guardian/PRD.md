# PRD: 燃气管线守护者

## 产品定位
第三方施工燃气管线保护的协同管理工具。施工方在线备案、监管方在线审核、燃气企业在线确认，全程留痕可追溯。

## 目标用户
- **监管所**：审核施工备案、监督现场交底、统计施工情况
- **燃气企业安全部**：接收施工通知、派员现场监护、确认交底
- **施工单位**：提交施工方案、查询管线信息、完成现场交底

## 核心功能（MVP）

### 1. 施工备案
- 施工单位填写施工备案表（项目名称、施工地点、施工时间、施工范围、管线保护方案、联系人）
- 支持现场拍照上传
- 提交后自动通知监管方和燃气企业

### 2. 审核管理
- 监管方在线审核（通过/退回/要求补充）
- 燃气企业确认收到通知
- 审核意见留痕

### 3. 现场交底确认
- 施工方/监管方/燃气企业三方现场确认
- 拍照+签名确认
- 生成交底确认单（可打印/可导出）

### 4. 统计看板
- 本月备案数、审核完成率、施工中项目数
- 近期施工一览表
- 历史记录搜索与导出

## 技术方案
- 单页HTML，浏览器打开即用
- 数据存localStorage
- Chart.js统计图表
- 拍照用浏览器Camera API
- 签名用Canvas
- 导出CSV

## 非功能需求
- 手机/电脑兼容
- 所有Chart.js调用try-catch保护
- 表单提交防重复
- 数据可备份导出

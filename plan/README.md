### 技术选型
- **前端**：保持单页 HTML（PWA），嵌入鉴权 SDK
- **后端**：Node.js + Express，部署到 Railway / Vercel
- **数据库**：PostgreSQL（Supabase 免费层）
- **支付**：LemonSqueezy（国内用户友好，支持支付宝/微信）
- **身份认证**：JWT + bcrypt，邮箱注册
- **托管**：后端 Railway，前端 GitHub Pages

## 📣 推广策略

### 目标客户画像
1. **区县级燃气监管所**（全国约 3000+ 个）
   - 痛点：纸质台账、手工统计、上级检查压力大
   - 决策人：所长、分管副局长
   
2. **燃气公司安全部门**（全国约 4000+ 家）
   - 痛点：入户安检量大、隐患跟踪难、合规要求高
   - 决策人：安全总监、部门经理

3. **第三方燃气检查机构**（全国约 500+ 家）
   - 痛点：多个项目并行、报告格式统一、客户交付
   - 决策人：机构负责人

### 推广渠道

#### 线上渠道（零成本）
| 渠道 | 方式 | 预期效果 |
|------|------|----------|
| 燃气行业微信群 | 直接分享工具链接，口碑传播 | 高转化，但依赖群主关系 |
| 行业论坛/知乎 | 发使用心得/教程帖 | 长尾流量，SEO |
| 微信公众号 | 写行业痛点文 + 工具推广 | 精准触达决策者 |
| 抖音/视频号 | 发工具使用演示短视频 | 覆盖年轻人 |
| 燃气行业展会/会议 | 现场演示 + 名片 | 高价值客户 |
| 百度 SEO | 优化网站关键词 | 被动获取客户 |

#### 线下渠道
| 渠道 | 方式 | 成本 |
|------|------|------|
| 行业会议/展会 | 展位/宣传单 | ¥500-2000 |
| 上门演示 | 预约拜访区县监管所 | 交通费 |
| 代理商 | 各地燃气设备代理商 | 分润 |

### 推广素材准备
- [ ] 产品介绍 PDF（一页纸）
- [ ] 2分钟产品演示视频
- [ ] 成功案例（虚构 MVP 验证期可暂不做）
- [ ] 免费版引流入口
- [ ] 行业报告/白皮书（用工具数据生成）

### 冷启动策略
1. **先免费后付费**：现有用户继续免费使用，新功能/云同步收费
2. **邀请有奖**：老用户邀请同行注册，赠送专业版时长
3. **行业大V推广**：联系燃气行业公众号/博主合作推广
4. **行业协会合作**：与地方燃气协会合作推荐

### 定价验证
- 首批 100 个专业版用户：¥49/年（早鸟价）验证付费意愿
- 根据付费率调整定价
